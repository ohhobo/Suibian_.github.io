### Introduction

- 在日常生活中，大学生往往对未来的职业充满困惑，感到迷茫或者焦虑。互联网上，职业相关话题引起热议，如知乎上“所学职业与未来工作不对口怎么办”的话题一度登上热搜榜单；越来越多人对职场定制服务表现出付费倾向，在淘宝上，简历代做月销上万。为了解决大学生、求职者职业焦虑、职场定位迷茫的痛点问题，职记APP 应运而生。
